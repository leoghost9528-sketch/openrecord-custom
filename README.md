## ✨ 定制功能

### 1. 摄像头大小调整
- 添加 4 档大小选择（Small/Medium/Large/X-Large）
- 实时调整，无需重启
- 位置：控制栏 ⊕ 按钮

### 2. 优化视觉效果
- 去除摄像头周围的蓝色声音边缘动画
- 提供更简洁专业的录制界面

### 3. Windows 启动脚本
- 一键启动开发服务器
- 中文界面，自动检查项目目录

## 🚀 快速开始

\`\`\`bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
\`\`\`

访问 http://localhost:3000

## 🛠️ 技术栈

- React 19 + TypeScript
- Vite 6
- Canvas API + MediaRecorder API
- Web Audio API
- Vercel 部署
